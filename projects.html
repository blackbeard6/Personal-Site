Projects

GreenLight

A native iOS app that utilizes OCR technology to scan Dartmouth Student IDs. The app is used to log the entrance of guests into student events, as well as provide security features including varifying that the student is not on the event blacklist. Won second place in the empowerment category of HackDartmouth VII in April of 2022.

BethHermbot

An artificial intelligence written in Python that plays chess. The AI utilizes Minimax search and is optimized with Alpha-Beta pruning, transposition tables, and lazy evaluation. Currently working on parallelizing the search and building a library of openings and endgame strategy. Has an ELO of roughly 1500.

Tiny Search Engine

A tiny search engine for Unix CLI. Uses crawling and indexing to print document sets in order of decreasing score (rank) according to tokenized user queries. Developed using C, Bash, and Makefile.

Hand Written Digit Classifier

Implemented an artificial neural network from scratch to classify handwritten digits with approximately 81.5% accuracy.
